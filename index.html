How It Works

Three steps. Two PCs.
Zero headache.

01

Scan

LazyTransfer reads your Windows registry and finds every installed program. Noise like Visual C++ redistributables and .NET runtimes is filtered out automatically.

02

Bundle

Your program list, user files, browser bookmarks, and settings are saved as a portable JSON bundle. Copy it to a USB drive or send it over your network.

03

Install

On the new PC, load the bundle and hit install. LazyTransfer uses winget or Chocolatey to silently reinstall everything with real-time progress tracking.

PROGRAMS

Program Migration

Smart matching maps your installed apps to package manager IDs. Fuzzy search + a built-in static map handles edge cases.

  • 10 app categories with toggle filters
  • winget preferred, Chocolatey fallback
  • Already-installed detection
  • Per-app progress with ETA
FILES

Files Migration

Copy Documents, Downloads, Pictures, Videos, Music, and Desktop folders. Export Chrome, Firefox, and Edge bookmarks as raw files.

  • ZIP archive or direct robocopy modes
  • Multithreaded copy (robocopy /MT:4)
  • Migration manifest for audit trail
  • Async folder size calculation
NETWORK

Remote Transfer

Three ways to move files over LAN without a USB drive. Start a web server, use direct TCP, or copy via shared folders.

  • HTTP server — download from any browser
  • Direct TCP with binary streaming
  • UNC shared folder via robocopy
  • Auto firewall rules
RESTORE

Full Restore

Point at any bundle folder and LazyTransfer auto-detects programs, files, system settings, and bookmarks inside.

  • Selective restore — pick what you want
  • Phased execution (settings → programs → files)
  • Bookmark restore with auto-backup
  • Error-resilient — continues on failure
